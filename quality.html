Our Approach

Quality is Not an Inspection Step — It's Embedded in Every Stage

From raw material selection through electrochemical deposition, washing, reduction, and classification, consistent powder properties are ensured through scientific control and statistical process monitoring.

Every batch undergoes comprehensive testing per Indian Standards (IS) and ASTM equivalents. Certificate of Analysis accompanies every shipment, providing complete transparency and traceability.

  • Every batch tested — no exceptions
  • Multiple operator verification for critical parameters
  • Statistical process control with control charts
  • Records retained for minimum 5 years
  • Complete traceability from raw material to shipment

Testing Protocol

Comprehensive Testing — Every Batch

01
Chemical Analysis

Method: IS 440 / ASTM E53 (Electrolytic Method). Specification: ≥99.80% Cu. Equipment: Electrolytic analysis / ICP-OES. Trace elements monitored: Fe, Pb, Ni, Sn, Ag, O₂.

IS 440ASTM E53
02
Apparent Density

Method: IS 4848 / ASTM B212. Equipment: Hall Flowmeter with calibrated 25 cm³ cup. Specification: Grade-specific (0.6–4.5 g/cm³). Multiple tests per batch for statistical confidence.

IS 4848ASTM B212
03
Hall Flow Rate

Method: IS 4840 / ASTM B213. Equipment: Hall Flowmeter Funnel (2.5mm orifice). Test: Time for 50g powder to flow. Specification: Non-flowing to 25–40 sec/50g (grade-specific).

IS 4840ASTM B213
04
Particle Size Analysis (Sieve)

Method: IS 460 / ASTM B214 / BSS 410. Equipment: Mechanical sieve shaker with ASTM E11 standard sieves. Sample: 100g. Sieving time: 15–20 minutes. Mesh sizes: 60, 100, 200, 325.

IS 460ASTM B214BSS 410
05
Morphological Analysis

Method: Light Optical Microscopy (LOM) / Scanning Electron Microscopy (SEM). Magnification: 100X to 5000X. Assessment: Dendritic character, aspect ratio, agglomeration check.

LOMSEM
06
Process Control Testing

Moisture content: Loss on drying (105°C, 2 hours) — Spec: <0.2%. Oxygen content: Combustion analysis — Typical: <0.3%. Statistical monitoring with control charts and ±10% limits.

MoistureOxygenSPC
Documentation

Certificate of Analysis — Every Shipment

Every shipment of Arti Metal copper powder is accompanied by a Certificate of Analysis (COA) providing complete transparency and traceability for your quality records.

COA Includes:

  • Batch/Lot Number (full traceability)
  • Manufacturing Date
  • Product Grade & Category
  • Chemical Analysis Results (Cu%, trace elements)
  • Apparent Density value (g/cm³)
  • Hall Flow Rate (if applicable)
  • Complete Sieve Analysis Data (all mesh fractions)
  • Particle Shape Assessment
  • Tested by: Sr. Engg. Quality
  • Approved by: Mr. A. Jain
